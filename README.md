# ☕ KRAFT: Theke unter Beschuss

Ein Tower-Defense-Spiel mit Berliner Kiez-Humor, inspiriert von [Kiez Kaffee Kraft](https://www.kiezkaffeekraft.de/).

Montag, 8:29 Uhr. Die Schlange reicht bis zur Straße. Verteidige die Theke gegen Hafermilch-Fanatiker, Influencer, das Finanzamt und den Gentrifizier — mit Siebträger, Banana Bread und dem KRAFT-Team.

## 🚀 Deploy auf GitHub Pages

1. Repo erstellen: [github.com/new](https://github.com/new) → `kraft-td` → Public
2. "Add a README file" ankreuzen → Create
3. "Add file" → "Upload files" → alle 6 Dateien hochladen → Commit
4. Settings → Pages → Deploy from branch → main / root → Save
5. Spielen: `https://DEINNAME.github.io/kraft-td/`

## 📱 Als App installieren

- **iOS**: Safari → Teilen → "Zum Home-Bildschirm"
- **Android**: Chrome zeigt "Installieren" Banner
